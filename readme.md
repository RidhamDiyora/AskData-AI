# 🤖 AskData AI

AI-powered Data Analyst that converts natural language into SQL queries using a local LLM and generates insights with visualizations.

---

## 🚀 Live Features

- 🧠 Natural Language → SQL (LLM-powered)
- 📊 Automatic Data Visualization
- 🔍 AI-generated Business Insights
- 🛡️ SQL Validation & Safety Layer
- 🔁 Auto Error Correction (Fixes broken queries)
- 💻 Runs locally (no API cost)

---

## 🛠 Tech Stack

- Python
- Streamlit
- SQLite
- Local LLM (Ollama + Llama3)
- Pandas & Matplotlib
